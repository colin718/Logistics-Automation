# 📦 스마트스토어 송장 자동 매칭 & 구독 관리 시스템 (Logistics Automation Pro)

### 📖 프로젝트 소개
스마트스토어 판매자들이 엑셀 작업으로 겪는 **송장 입력 실수와 누락 문제를 해결**하기 위해 개발한 자동화 프로그램입니다.
**실제 셀러들에게 배포**하여 피드백을 반영했으며, 불법 사용 방지를 위한 **라이선스 서버(구글 시트 연동)**까지 구축했습니다.

### 🛠️ 사용 기술 (Tech Stack)
* **Language:** Python 3.9
* **GUI:** Tkinter (CustomTkinter)
* **Data:** Pandas (Excel Parsing & Mapping)
* **Security:** Google Sheets API (License Auth), Threading

### ✨ 주요 기능
1.  **원클릭 자동 매칭:** 네이버 주문서와 택배사 엑셀을 비교하여 송장 번호 자동 입력
2.  **누락 방지 시스템:** 배송되지 않은 고객 명단을 별도 엑셀로 자동 추출
3.  **구독형 라이선스 관리:** 구글 스프레드시트를 DB로 활용하여 실시간 사용 권한 제어

### 💡 문제 해결 경험
* **문제:** 대용량 엑셀 처리 시 GUI가 멈추는 '프리징 현상' 발생
* **해결:** 데이터 처리 로직을 별도 스레드(Threading)로 분리하여 UI 반응성 확보
